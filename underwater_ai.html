Methodology

1. Training Phase (Server)
  • ‚Ä¢ Train UDnet using unpaired data from EUVP, UIEB, Underwater ImageNet
  • ‚Ä¢ Generate pseudo-paired data for LU2net & MobileIE
  • ‚Ä¢ Validate with CLIP & YOLO to remove bad samples
  • ‚Ä¢ Custom Indian Ocean Dataset (IOD) for regional specificity
2. Deployment Phase (Edge)
  • ‚Ä¢ Run MobileIE/LU2net for frame enhancement on Jetson devices
  • ‚Ä¢ Feed enhanced frames to YOLO detector for real-time analysis
  • ‚Ä¢ Send telemetry and data back to central server
3. Retraining Loop
  • ‚Ä¢ Nightly retraining using new ROV-collected data
  • ‚Ä¢ CLIP-based validation filters poor samples
  • ‚Ä¢ Automated weight distribution to deployed devices
